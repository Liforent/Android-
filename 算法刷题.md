# 复杂度

## 时间复杂度

### 时间复杂度从优到劣

1. O(1) 常数阶

   数组访问

2. O(logN) 对数阶

   二分查找

3. O(n) 线性阶

   简单查找

4. O(n*logN) 线性对数阶

   常见于高效排序算法，快速排序，归并排序

5. O(n^2) 平方阶

   双重循环，冒泡排序，选择排序

6. O(2^n) 指数阶

   极其低效，递归斐波那契

7. O(n!) 阶乘阶

   最差情况，旅行商问题暴力解

### 时间复杂度计算规则

- **忽略常数项**：O(2n) → O(n)
- **只保留最高阶**：O(n² + n) → O(n²)
- **乘法规则**：嵌套循环复杂度相乘

## 空间复杂度

空间复杂度衡量算法执行过程中临时占用的存储空间大小。

### 常见空间复杂度

1. O(1) 常数空间

   只使用固定数量的变量

2. O(n) 线性空间

   需要额外数组存储N个元素

3. O(n²) 平方空间

   二位数组

### 空间复杂度分析要点

- 包括输入空间和辅助空间
- 通常关注辅助空间（额外开辟的内存）
- 递归调用会占用栈空间
